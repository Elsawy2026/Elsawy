# 💧 اللحيدان لتوريد المياه - ALLHIDAN FOR WATER SUPPLY

صفحة هبوط احترافية ومتقدمة لشركة اللحيدان لتوريد المياه مع تصميم عصري وتأثيرات بصرية مذهلة.

## ✨ المميزات

- 🎨 **تصميم عصري وجذاب** - واجهة مستخدم احترافية مع ألوان متناسقة
- 📱 **متجاوب بالكامل** - يعمل بشكل مثالي على جميع الأجهزة (Mobile, Tablet, Desktop)
- ⚡ **تأثيرات بصرية متقدمة** - Animations سلسة وتأثيرات hover مميزة
- 🚀 **أداء سريع** - محسّن للأداء والسرعة
- 🌊 **تأثيرات الموجات** - خلفيات متحركة وتأثيرات parallax
- 🎯 **سهل الاستخدام** - تجربة مستخدم ممتازة
- 🌐 **دعم اللغة العربية بالكامل** - تصميم RTL احترافي

## 🚀 الاستخدام

### الطريقة السريعة
افتح ملف `index.html` مباشرة في المتصفح

### باستخدام سيرفر محلي

**باستخدام Python:**
```bash
cd landing-page
python -m http.server 8000
```

**باستخدام Node.js (http-server):**
```bash
cd landing-page
npx http-server -p 8000
```

**باستخدام PHP:**
```bash
cd landing-page
php -S localhost:8000
```

ثم افتح المتصفح على: `http://localhost:8000`

## 🎨 التخصيص

### الألوان
يمكنك تخصيص الألوان من ملف `style.css` عن طريق تغيير متغيرات CSS:

```css
:root {
    --primary-color: #0ea5e9;      /* اللون الأساسي */
    --secondary-color: #06b6d4;    /* اللون الثانوي */
    --accent-color: #f97316;       /* لون التمييز */
    /* ... المزيد من الألوان */
}
```

### المحتوى
قم بتعديل محتوى الصفحة من ملف `index.html` حسب احتياجاتك.
